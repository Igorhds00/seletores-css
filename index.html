Seletores CSS

Aplicando CSS em Diferentes Partes do HTML

1. O seletor de elementos segmenta com base no nome da tag HTML.

2. Os seletores de classe direcionam elementos com base no valor do atributo de classe.

3. Os seletores de ID direcionam os elementos com base no valor do atributo id.

4. Os seletores de atributos direcionam elementos com base em seus atributos e valores.

5. O seletor universal tem como objetivo estilizar todos os elementos.
